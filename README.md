# Phonebook Backend

This is a full-stack application built for Part 3 of the Full Stack Open course.

**Live Demo:** [https://phonebook-backend-natasha.onrender.com](https://phonebook-backend-natasha.onrender.com)

## Features
- Serves a React frontend from a Node/Express backend.
- Provides a RESTful API for managing phonebook entries.
- Uses Morgan for logging.

## How to run locally
1. Run `npm install`
2. Run `npm run dev` to start the server.
